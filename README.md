# 241RunnersAwareness.org

**241Runners Awareness** is a full-stack application created to reduce the number of missing and runaway special needs individuals. This project features a responsive public website and a secure, cloud-connected backend that stores critical emergency and biometric data for fast response and recovery.

---

## 🌐 Live Website  
[https://241runnersawareness.org](https://241runnersawareness.org)

---

## 💻 Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: C#, ASP.NET Core Web API  
- **Database**: Azure SQL  
- **Hosting**: GitHub Pages (Frontend), Azure App Service (Backend)  
- **Tools**: Swagger UI, GitHub, Visual Studio, VS Code
