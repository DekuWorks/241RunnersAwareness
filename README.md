# 241RunnersAwareness.org

**241Runners Awareness** is a full-stack application built to help reduce the number of missing and runaway special needs individuals. It features a responsive public website and a secure, cloud-based backend database to collect emergency information and biometric data for fast recovery and identification.

## 🌐 Live Website

## 📦 Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: C#, ASP.NET Core Web API  
- **Database**: Azure SQL  
- **Cloud Hosting**: GitHub Pages, Azure  
- **Tools**: Swagger, Visual Studio, VS Code, GitHub
